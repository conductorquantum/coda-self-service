# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.2.0] - 2026-04-06

### Added

- Connectivity reporting in heartbeat and connect payloads.
- `HeartbeatRejectedError` when the platform rejects a heartbeat.

## [0.1.0] - 2026-04-01

### Added

- Initial release: JWT auth, VPN management, Redis job consumer, webhook delivery, heartbeat, IR/QASM support, `JobExecutor` protocol, daemon mode, and CLI.
